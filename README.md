**Project: Loan Management Application**  

**Overview**  
Loan Management System is designed to handle the complete lifecycle of loan processing in a bank.  
It helps manage customer data, loan approvals, EMI tracking, and repayment schedules.

---

**Objective**  
To provide a seamless system for managing loan applications, approvals, disbursement, repayment, and recovery processes.

---

**Data Flow**  
Lead → Loan Application → Credit Check → Loan Disbursement →  
Repayment Schedule → Loan Payment → Loan Closure →  
Loan Collection (if default) → Loan Reports
